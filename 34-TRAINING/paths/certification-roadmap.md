# Certification Roadmap
### rmsecurity | 34-TRAINING

---

## Priority Path for rmsecurity Operator

```
Foundation → Offensive Core → Specializations
```

### Foundation (complete first if not already certified)

| Cert | Provider | Cost | Duration | Priority |
|------|---------|------|---------|---------|
| CompTIA Security+ | CompTIA | ~$400 | 2-3 months | Entry-level baseline |
| eJPT | eLearnSecurity | $200 | 1-2 months | Practical intro to pentesting |

### Offensive Core (primary focus)

| Cert | Provider | Cost | Duration | Value |
|------|---------|------|---------|-------|
| **OSCP** | OffSec | $1,499 (90 days lab) | 3-6 months | Gold standard — opens doors, proves practical skill |
| **CRTP** | Altered Security | $249 | 1-2 months | AD attacks — directly applicable to 80% of engagements |
| **eWPT** | eLearnSecurity | $200 | 1-2 months | Web app pentesting certification |
| **BSCP** | PortSwigger | Free (exam $149) | 2-3 months | Web app — Burp Suite focused |

### Cloud Specializations

| Cert | Provider | Cost | Duration | Value |
|------|---------|------|---------|-------|
| AWS Security Specialty | AWS | $300 exam | 3-4 months | Required for credible AWS assessments |
| CCSP | (ISC)² | $599 exam | 4-6 months | Cloud security — recognized by enterprise clients |
| AZ-500 | Microsoft | $165 exam | 2-3 months | Azure security — M365/Azure assessments |

### Advanced / Red Team

| Cert | Provider | Cost | Duration | Value |
|------|---------|------|---------|-------|
| OSED | OffSec | $1,299 | 3-6 months | Exploit development |
| CRTO | Zero-Point Security | $430 | 2-3 months | Red team ops, C2 |
| OSEP | OffSec | $1,299 | 3-6 months | Evasion techniques |

---

## Recommended Study Order

**Year 1 (establish credibility):**
1. OSCP — non-negotiable for a pentest firm
2. CRTP — immediate ROI in AD engagements

**Year 2 (expand service lines):**
3. BSCP or eWPT — deepen web app capability
4. AWS Security Specialty or AZ-500 — unlock cloud service line

**Year 3+ (differentiation):**
5. CRTO — red team engagements
6. CCSP — enterprise sales credential
