# 25-BLUE-TEAM — Defensive Security Advisory
### rmsecurity | CCOS

## Purpose

Advisory playbooks for helping clients improve their defensive posture:
detection engineering recommendations, security hardening, and security
architecture review. Delivered as findings or as separate advisory engagements.

## Directory Structure

```
25-BLUE-TEAM/
├── README.md
└── playbooks/
    └── security-hardening.md    <- CIS benchmark-based hardening guidance
```

## Service Scope

This domain supports:
1. **Remediation guidance** — technical hardening advice in pentest reports
2. **Hardening reviews** — comparing client config against CIS Benchmarks
3. **Detection recommendations** — SIEM rules and alerting for findings we exploit

## Related Domains

- `22-PENTESTING/` — blue team recommendations follow from pentest findings
- `43-REMEDIATION/` — hardening is the long-term fix for many pentest findings
